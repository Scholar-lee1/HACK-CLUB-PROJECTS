# 🚀 Future Me Judge

See your future before you choose.

---

## 🧠 About
Future Me Judge is a decision simulator that helps users explore the possible outcomes of their choices.  
Enter a decision and get three structured future paths:
- ✅ Best Case  
- ⚠️ Worst Case  
- 🤯 Unexpected Outcome  

---

## ✨ Features
- Clean and responsive UI  
- Real-time decision simulation  
- Structured future scenarios (timeline + outcomes)  
- Fast and interactive experience

## 🚀 How It Works
1. Enter a decision  
2. Click “Simulate”  
3. Instantly view 3 possible futures
